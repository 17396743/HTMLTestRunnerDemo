# HTMLTestRunner 测试文档生成脚本（标准版/纯文字）

使用方式（例子）：
```python
 HTMLTestRunner(
        stream=f,
        title="单元测试",
        description="测试一期",
        verbosity=2
    ).run(suite) 
```
